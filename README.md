# 🐍 Snake Game Pro

A modern and responsive Snake Game built using **HTML, CSS, and JavaScript**. The game features smooth gameplay, level progression, high-score tracking, sound effects, and a mobile-friendly interface.

## 🚀 Features

* 🎮 Classic Snake Gameplay
* 📈 Level Progression System
* 🏆 High Score Tracking using Local Storage
* 🔊 Sound Effects
* 📱 Responsive Design
* ⏸️ Pause Functionality
* 🔄 Restart Option
* 💻 Keyboard Controls
* 🎯 Random Food Generation
* 💥 Game Over Screen

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎮 Controls

| Key   | Action         |
| ----- | -------------- |
| Enter | Start Game     |
| ↑     | Move Up        |
| ↓     | Move Down      |
| ←     | Move Left      |
| →     | Move Right     |
| Space | Pause / Resume |

## 🏆 High Score System

The game stores the highest score using the browser's Local Storage, allowing players to keep track of their best performance even after refreshing the page.

## 📱 Responsive Design

The game board automatically adjusts to different screen sizes, making it playable on desktops, tablets, and mobile devices.
